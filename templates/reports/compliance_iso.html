1. Introduction to ISO 27001 Context

The ISO/IEC 27001 standard outlines requirements for establishing, implementing, maintaining, and continually improving an Information Security Management System (ISMS). VulnVision assists the organization in managing technical vulnerabilities (Annex A.12.6.1) and maintaining secure system engineering principles (Annex A.14.2.5).

2. Annex A Control Mapping

Vulnerabilities discovered typically represent a failure in one or more Annex A controls:

  • A.12.6.1 Management of technical vulnerabilities: Failure to identify or patch vulnerabilities in a timely manner.
  • A.14.2.5 Secure system engineering principles: Insecure application configuration or deployment practices.
  • A.13.1.2 Security of network services: Misconfigured or outdated network services exposed to adversaries.
